## What VULK Generates

### 8 Platform Targets

| Platform | Stack | Output |
|----------|-------|--------|
| **Web Apps** | React + Vite + Tailwind CSS | Full web applications with routing, state, auth |
| **Flutter** | Dart + Riverpod + GoRouter | Native mobile apps with Clean Architecture |
| **React Native** | Expo SDK 52+ + NativeWind | Cross-platform mobile with Expo Router |
| **Shopify Themes** | Liquid + Online Store 2.0 | Complete themes with settings schema + live preview |
| **Shopify Apps** | Remix + App Bridge + Polaris | Full apps with OAuth, webhooks, billing |
| **Shopify Hydrogen** | Remix + Storefront API | Headless commerce storefronts |

| **Browser Games** | Three.js + Canvas | 2D/3D games with physics engines |

No other AI builder supports more than one.

### What Makes It Different

- **76 premium components** from Aceternity UI and Magic UI cached and injected into generations
- **15+ AI models** — Claude Opus 4.6, GPT-5.2, Gemini 3.1 Pro (1M context), Grok, DeepSeek, Llama
- **Auto-generated backends** — PostgreSQL schemas, JWT auth, CRUD APIs from prompt analysis
- **Live Shopify preview** — Built-in Liquid rendering engine with 40+ filters and mock store data
- **3D model generation** — Image to GLB via Replicate Trellis 2 with inline viewer
- **Voice editing** — Edit projects by talking via Gemini Live bidirectional streaming
- **Figma import** — Paste a Figma URL, get React + Tailwind code
- **5 deploy targets** — Cloudflare, Vercel, Netlify, GitHub, auto-deploy with custom domains
- **Full code ownership** — Export as ZIP, push to GitHub, self-host anywhere

## Shopify Development (New)

Vulk is the first AI builder to generate production-ready Shopify code. Three modes, auto-detected from your prompt:

1. **Liquid Themes** — Dawn-compatible Online Store 2.0 themes with settings schema, mobile-first CSS, ARIA accessibility
2. **Shopify Apps** — Remix + App Bridge + Polaris with OAuth, Admin GraphQL API, billing, webhooks
3. **Hydrogen Storefronts** — Headless commerce with Storefront API, cart operations, SEO

Live Liquid preview with 40+ custom filters and full mock store context (12 products, collections, cart, navigation).

Read more: [Vulk Introduces Shopify Development with AI](https://vulk.dev/blog/vulk-introduces-shopify-development-with-ai)
